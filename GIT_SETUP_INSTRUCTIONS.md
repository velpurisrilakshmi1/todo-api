# Commands to Run After Creating GitHub Repository

## After you create the repository on GitHub, run these commands:

```bash
# Add the GitHub repository as remote origin
git remote add origin https://github.com/velpurisrilakshmi1/todo-api.git

# Push your code to GitHub
git push -u origin main
```

## If you get authentication errors, try:

```bash
# For HTTPS authentication (will prompt for username/password or token)
git push -u origin main

# Or configure your credentials first:
git config --global credential.helper store
```
